# PlainTextConverterApp

PlainTextConverterApp is a lightweight, cross‑platform Java 21 desktop utility that converts **plain English text** into structured formats instantly. It’s designed to be sleek, modern, and easy to use, with built‑in onboarding and helpful actions for every output.

---

## ✨ Features
- **Plain English → Structured Data**  
  Convert simple text like `Name: Alice, Age: 12, City: New York` into:
  - JSON  
  - XML  
  - CSV  
  - JSONL  

- **Action Buttons for Each Format**  
  - Copy Data → one‑click copy to clipboard  
  - Download Data → save as `.json`, `.xml`, `.csv`, `.jsonl`  
  - Send Data → POST to any API endpoint with optional key  

- **Modern UI**  
  - Sleek Nimbus‑based theme with clean fonts and borders  
  - Tabbed outputs for easy navigation  
  - Split‑pane layout for input vs. results  

- **Onboarding Experience**  
  - “How to Use” dialog shown on first launch  
  - Skip button for quick access  

- **Offline & Portable**  
  - Runs entirely offline (except “Send Data”)  
  - Distributed as a single Fat JAR — no extra dependencies
